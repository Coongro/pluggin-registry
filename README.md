# Coongro Plugin Registry

Catálogo de plugins disponibles para Coongro.

## Uso

Claude Web lee `plugins-catalog.yaml` para conocer el ecosistema de plugins antes de diseñar nuevos.

URL directa:
```
https://raw.githubusercontent.com/Coongro/pluggin-registry/main/plugins-catalog.yaml
```

## Actualizar

Editar `plugins-catalog.yaml` cuando se publique un nuevo plugin.
